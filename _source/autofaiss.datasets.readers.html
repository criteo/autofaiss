autofaiss.datasets.readers.local_iterators module¶

Functions that read efficiently files stored on disk

autofaiss.datasets.readers.local_iterators.read_embeddings_local(local_embeddings_path, batch_size=None, verbose=True, reg_exp_pattern='.+\\\\.npy')[source]¶

Iterate over embeddings arrays saved on disk. It is possible to iterate over batchs of files and yield stacked embeddings arrays.

The implementation adopted here is chosen for memory concern: it is very important for autofaiss to avoid using more memory than necessary. In particular, for the faiss training, a large embedding array is nessary. It is not possible to save it twice in memory. This implementation pre-allocate an array of size batch_size and keep it updated during the iteration over files. The maximum memory usage is batch_size * dim * 4

Parameters

  • local_embeddings_path (str) – Path on local disk of the embedding in numpy format.

  • batch_size (int (default None)) – Outputs a maximum of batch_size vectors, the default is the size of the first file This parameter is useful when working with many small files.

  • verbose (bool) – Print detailed informations if set to True

Returns

embeddings_iterator – An iterator over batchs of stacked embedding arrays.

Return type

Iterator[np.ndarray]
